# Tongkat Sakti - Installation Guide

## 📥 How to Install

### Method 1: Direct Download (Recommended)
1. Go to: https://github.com/mwehehe501-collab/Upgrate-Addon/releases
2. Download: Tongkat-Sakti-v1.0.0.mcaddon
3. Double-click the file
4. Done!

### Method 2: Manual
- Windows: %APPDATA%\\Microsoft\\Windows\\Start Menu\\Programs\\Minecraft Launcher\\com.mojang\\minecraftuserdata\
- Copy .mcaddon file there

## 🎮 How to Use

### Open Menu
- Right-click (use) Tongkat Sakti

### Duplicate Modes
**Inventory Mode**: Select item → Duplicate
**World Mode**: Look at object (10 blocks) → Select → Duplicate

## 🐛 Troubleshooting

### Add-on not appearing?
- Check if enabled
- Restart Minecraft
- Reinstall if needed

### Menu not opening?
- Make sure RIGHT-clicking (use button)
- Hold Tongkat Sakti in hand

## ✅ Requirements
- Minecraft Bedrock 1.20.0+
- All platforms supported
- Achievements work!
