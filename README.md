# Tongkat Sakti v1.0.0

## 🎯 Minecraft Bedrock Add-on

### ✨ Fitur Utama
- ✅ **Unlimited Duplicate** - Tanpa cooldown!
- ✅ **Inventory Mode** - Duplikat item dari inventory
- ✅ **World Mode** - Duplikat blok/entitas dalam 10 blok
- ✅ **Smart Stacking** - Auto tumpuk item
- ✅ **Auto-spawn** - Muncul otomatis saat spawn
- ✅ **Achievement Support** - Full compatible
- ✅ **No Experimental** - Vanilla friendly

### 🚀 Installation
1. Download: `Tongkat-Sakti-v1.0.0.mcaddon`
2. Double-click untuk install
3. Enable di world settings
4. Play!

### 📖 Documentation
- **INSTALLATION.md** - Setup guide
- **TECHNICAL.md** - Architecture details
- **BUILD_AND_RELEASE.md** - Build instructions

### 🎮 How to Use
1. Right-click Tongkat Sakti
2. Choose mode: Inventory or World
3. Select item/block/entity
4. Duplicated!
